# Proyek 3 Mobile

Aplikasi pelanggan berbasis Flutter untuk belanja hijab dengan fitur rekomendasi warna berbasis AI.

## Fitur
- Login pengguna
- Melihat produk hijab
- Rekomendasi warna sesuai warna kulit (AI)

## Teknologi
- Flutter (Mobile App)
- Laravel (Backend API)
- AI Recommendation System
